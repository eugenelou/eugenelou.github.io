Sources for Android API xx Platfrom not found

最近忙着项目上线，死丢丢已经一段时间没有打开了，今天把死丢丢升级到2.1之后，发现无法关联源码。赶紧上StackOverFlow搜索了下，果然已经有人踩坑了。Sources for Android API 23 Platfrom not found

解决方法

找到死丢丢的jdk.table.xml配置，通过你的api level版本找到对应配置文件，在sourcePath节点下的composite中添加<root type="simple" url="file://$USER_HOME$/Documents/sdk/sources/android-23" />。(我的是23)

重启死丢丢问题完美解决。
最近开始要重抄旧业了，准备重新温习下基础部分，希望能在android开发的道路上走的更远。
